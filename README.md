# Lync
a url-shortener made with FastAPI

**Visit site: https://745368.xyz**

##  Getting Started
1. Clone the repository:
    ```bash
    git clone https://github.com/silentfin/url-shortener.git
    cd url-shortener
    ```

2. Install dependencies:  

    **Using uv (recommended):**
    ```bash
    uv sync
    ```

3. Run in developement mode:
    ```bash
    fastapi dev main.py
    ```
